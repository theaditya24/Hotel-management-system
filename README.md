🏨 Hotel Management System

A Hotel Management System designed to efficiently manage hotel operations such as customer registration, room booking, booking management, and payment processing. This system helps streamline hotel administration by maintaining organized records of customers, rooms, and bookings.

The project demonstrates the implementation of core programming concepts, structured data management, and modular system design to solve real-world hospitality management problems.

📌 Features
👤 Customer Management

    - Add new customers
    
    - Search customers by Name
    
    - Search customers by Phone Number
    
    - View customer details
    
    - Maintain customer records

🏨 Room Management

    - Add new rooms
    
    - View available rooms
    
    - Track room occupancy status
    
    - Manage room details

📅 Booking Management

    - Create new bookings
    
    - Assign rooms to customers
    
    - Manage check-in and check-out
    
    - View booking details

💳 Payment Management

    - Record payments
    
    - Track payment status
    
    - Generate payment records

🔎 Search Functionality

Search customers quickly using:
    - Phone Number

    - Name

    The system starts with a main menu.
